electsys-api
============

上海交通大学新版教学信息服务网 API。

|image0| |image1| |image2|


功能
====

- ``jAccount`` 登录
- 查询校历
- 查询课程表
- 查询考试安排
- 查询考试成绩

`以下操作仅开放选课时段可用`

- 模糊／精确搜索课程库
- 查询已选课程
- 选课
- 退课
